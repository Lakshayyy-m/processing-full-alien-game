# Project Requirements - UI Design and Programming

## Overview
To achieve full marks, the project **MUST** include all 8 requirements listed below.

---

## 1. Java Generics and Data Structures
- **(a)** Use `ArrayList` (Java Generic) for the graphics application
- **(b)** Use a static array

---

## 2. Object-Oriented Design & Hierarchy
- **(c)** Design separate class files
- **(d)** Build multiple parent-child hierarchical objects
  - Must incorporate **abstract classes**
  - Must incorporate **interface classes**

### Hierarchy Requirements:
- Create **minimum 3-level** grandparent-parent-child inheritance hierarchy
- Must have **minimum 5 different shaped objects**

### Object Types Required:
1. **One human figure** (with hierarchy)
2. **Two additional hierarchical objects**, which could be:
   - Animal
   - Alien creature
   - Natural object
   - Building
   - Environment (e.g., car, tree with leaves)

---

## 3. Stack Structure for Transformations
- **(e)** Design and apply **push and pop stack structure**
- Properly associate hierarchical objects
- Build gestures for **2D transformation**

---

## 4. Custom Movement Functions
- **(f)** Design multiple custom **function/method blocks**
- Add unique movement to parent-child based objects

---

## 5. User Interactions
- **(g)** Implement **mouse interaction**
- **(h)** Implement **keyboard interaction**
- Both should transform hierarchy-chained objects
- Example: Rotating hierarchical objects expressing different motion

---

## 6. Image Filter
- **(i)** Add a **custom image filter** (similar to Lab03)

---

## 7. User Interface Components
- **(j)** Add **minimum 4 UI components**

### UI Component Requirements:
- **2 components** must be **custom UI created from scratch**
  - Similar to Lab03 RGB slider with custom shapes
- **2 components** can be used from **Processing samples**

### Example UI Components:
- Rollover
- Button
- Handles
- Scrollbar
- Custom sliders
- Toggle switches

---
